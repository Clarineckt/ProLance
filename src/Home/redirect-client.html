Mobile App UI

11 hours ago


                About this project
                We are developing a mobile productivity app (iOS & Android) focused 
                on daily task management, habit tracking, and progress visualization. 
                We are seeking a skilled UI designer to create a clean, intuitive, and 
                modern interface that enhances user engagement and ease of use.

                Key Deliverables:
                - UI designs for ~10 core screens (e.g., Home, Task List, Calendar View)
                - Style guide (color palette, typography, buttons, icons)
                - Design system components for consistency
                - Prototypes (using Figma or Adobe XD)
                - Handoff-ready assets for development

                Design Goals:
                1. Minimalist, modern, and mobile-first layout
                2. Clear navigation and user flow
                3. Consistent use of whitespace and color for focus
                4. Dark mode (optional but preferred)

                Requirements
                1. Proven experience designing mobile app UI (portfolio required)
                2. Proficiency in Figma, Adobe XD, or similar tools
                3. Understanding of mobile UX best practices
                4. Ability to create reusable components and design systems
                5. Good communication and openness to feedback
                6. Bonus: Experience collaborating with developers

                Salary: $150
